# Prerequisites:
## If you are using docker:  
+ docker  
+ docker with compose or docker-compose
## Without docker:  
+ node  
+ yarn


# Run app
## 1. Install packages:
`yarn`

## 2. Start app

+ **With docker**  
run one of:  
`docker-compose up --build`  
`docker compose up --build`  
app:  
`localhost:5000`

+ **Without docker**  
run:  
`yarn run start`  
app:  
`localhost:3000`
